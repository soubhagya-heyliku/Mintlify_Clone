## About the Project

This is a **static landing page clone** inspired by the official Mintlify website UI.

What this project includes:
- Full landing page structure
- Multiple sections like hero, brands, features, customers, CTA, and footer
- Realistic SaaS-style layout
- No JavaScript used

The goal was not to make it functional, but to **replicate the design and structure** as close as possible.
